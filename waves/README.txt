PROSES PEREKAMAN DATA SUARA
-----------------------------
1. Rekamlah suara untuk kata-kata yang muncul di /wlist. Suara disimpan dalam
   format .wav.
2. Buatlah label untuk suara yang anda rekam, sesuaikan nama file dengan label.

format label adalah S<nomor kalimat> <kalimat>

contoh label
S0001 WOODLAND SMITH TWO FOUR JULIAN
S0002 FIVE FOUR NINE PHONE ZERO
S0003 THREE DIAL STEVE PHIL NINE
S0004 SEVEN PHONE THREE SIX SMITH
...

Ini artinya terdapat sebuah file rekaman dengan nama "S0001.wav" dengan isi 
WOODLAND SMITH TWO FOUR JULIAN. Terdapat file rekaman dengan nama "S0002.wav"
dengan isi FIVE FOUR NINE PHONE ZERO.

Seperti yang terlihat, sesungguhnya sebuah kalimat boleh tidak sesuai dengan
grammar yang ada.

3. Buatlah sebuah folder disini dan upload hasil rekaman di dalam folder anda.
